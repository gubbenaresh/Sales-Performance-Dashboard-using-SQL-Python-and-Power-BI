# Interview Explanation

## Project Name

Sales Performance Dashboard using SQL, Python and Power BI

## Project Overview

This project analyzes sales data using Python, SQL, and Power BI. I cleaned the raw sales dataset using Python, performed exploratory data analysis, wrote SQL queries for business analysis, and created an interactive Power BI dashboard.

## Tools Used

- Python
- Pandas
- Matplotlib
- SQL Server
- Power BI
- GitHub

## Project Workflow

1. Downloaded sales dataset
2. Cleaned data using Python
3. Created new columns like year, month, quarter, profit margin, and shipping days
4. Performed EDA using Pandas and Matplotlib
5. Imported cleaned data into SQL Server
6. Wrote SQL queries for sales, profit, customer, product, category, region, and discount analysis
7. Built an interactive Power BI dashboard
8. Documented business insights

## Business Questions Answered

1. What is the total sales?
2. What is the total profit?
3. What is the average order value?
4. Which category generates the highest sales?
5. Which region performs best?
6. Which products generate the highest sales?
7. Which products are loss-making?
8. Which customers contribute most to sales?
9. How does discount affect profit?
10. What is the monthly sales trend?

## Key Learnings

This project helped me understand the complete data analysis workflow from raw data cleaning to dashboard creation. I learned how Python, SQL, and Power BI are used together in real business reporting.

## Simple Interview Answer

I built a Sales Performance Dashboard project using Python, SQL, and Power BI. I cleaned the sales dataset using Python and created useful columns like order year, month, profit margin, shipping days, and sales category.

Then I used SQL to analyze KPIs such as total sales, total profit, average order value, category performance, region performance, top products, top customers, and discount impact.

Finally, I created a Power BI dashboard with KPI cards, charts, slicers, and business insights. This project helped me understand how raw sales data can be converted into useful business reports.
