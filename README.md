# Sales Performance Dashboard using SQL, Python and Power BI

## Project Overview

This project analyzes sales performance using Python, SQL, and Power BI. The goal is to clean raw sales data, perform exploratory data analysis, write SQL queries, and create an interactive dashboard to understand business performance.

## Objective

The objective of this project is to analyze:

- Total sales
- Total profit
- Total orders
- Total customers
- Average order value
- Category-wise sales and profit
- Region-wise sales and profit
- Monthly sales trend
- Top products
- Top customers
- Discount impact on profit
- Segment-wise performance
- Shipping mode performance

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL Server
- SQL Server Management Studio
- Power BI
- GitHub

## Dataset

The project uses a sales dataset containing order, customer, product, sales, profit, discount, quantity, region, and shipping information.

## Project Workflow

1. Data collection
2. Data cleaning using Python
3. Feature creation
4. Exploratory data analysis
5. SQL database creation
6. SQL business analysis
7. Power BI dashboard creation
8. Business insights documentation
